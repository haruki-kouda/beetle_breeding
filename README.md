# README

# アプリケーション名
beetle_breeding
## アプリケーション概要
飼育情報を投稿でき、投稿内容を編集・削除できる。また現段階では外部サイトのスプレッドシートを使用してより細かな飼育管理が可能です。

## 利用方法
ユーザー登録を行い、飼育情報の登録を行います。投稿した飼育情報を飼育状況により、編集・削除が可能です。
## 目指した課題解決
カブトムシ・クワガタを飼育している方が飼育種の情報と画像をまとめられるようアプリケーションがなかったので飼育情報を整理できるようなアプリケーションを作成しました。
